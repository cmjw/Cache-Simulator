Cache Simulator
CS 378


To Run All Traces:
$ ./run.sh

To Run:
$ make
$ ./cache_simulator <trace>

To run with minimal output other than stats:
$ ./cache_simulator <trace> -n

To set associativity:
$ ./cache_simulator <trace> -n -a <associativity>

Input:
Trace file in .din Dinero 3 format.
